## Text to Image Generator – Complete Guide

This Text to Image Generator is a powerful web-based tool that allows you to instantly convert text into visually appealing images. It supports multiple customization options like font size, colors, dimensions, and padding, making it perfect for creating social media posts, thumbnails, banners, and even favicons.

---

### How to Use

Using this tool is quick and straightforward:

1. **Enter Your Text**
   Type or paste your desired text into the input box. You can also add multiple lines.

2. **Customize Appearance**
   Adjust different settings such as:

   * Font size
   * Image width and height
   * Padding
   * Text color
   * Background color

3. **Preview in Real-Time**
   The image updates automatically as you change settings.

4. **Download Your Image**
   Use the download option to export your image in:

   * PNG format
   * SVG format
   * ICO (Favicon) format
