## Scientific Calculator – Complete Guide

This scientific calculator is a powerful web-based tool designed to handle both basic and advanced mathematical calculations and efficiently. It is ideal for students, professionals, and anyone who needs quick and accurate results without installing any software.

### How to Use

Using the calculator is very simple:

1. **Enter Numbers & Operators**
   Click on the buttons to input numbers and operations like `+`, `-`, `*`, `/`.

2. **Switch Modes**

   * Use **Normal Mode** for basic calculations
   * Enable **Scientific Mode** to access advanced functions like `sin`, `cos`, `log`, `√`, etc.

3. **Select Angle Unit**
   Toggle between **Degrees** and **Radians** for trigonometric calculations.

4. **Calculate Result**
   Press the `=` button to get the final answer.

5. **View & Manage History**

   * Enable "Show History" to see previous calculations
   * Delete individual entries or clear all history
