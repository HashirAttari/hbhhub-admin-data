### **What is Hashing?**

Hashing is a process of converting any input (like text, passwords, or files) into a fixed-length string called a **hash**. Each hash algorithm has its own method, producing unique results even for small changes in input.

Our tool supports **80+ algorithms** across popular libraries such as `crypto-js`, `blakejs`, `adler32`, `hash-wasm`, `js-sha3`, and more.
