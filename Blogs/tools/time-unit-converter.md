## Time Converter – Complete Guide

The **Time Converter** is a versatile tool designed to help you quickly convert between different units of time, ranging from milliseconds to centuries. Whether you’re a student, engineer, developer, or just curious about time measurements, this converter makes calculations fast and accurate.

---

### How to Use

1. **Enter a Value**
   Type the time value you want to convert, for example, `120` or `3600`.

2. **Select the Input Unit**
   Choose the unit for the value you entered, such as seconds, minutes, or hours.

3. **View Results**
   The converter automatically calculates the equivalent value across all supported units.

4. **Change Units or Value**
   Simply modify the input value or select a different unit to see real-time updates.

---

### How It Works

This converter uses a **base unit system (seconds)**:

1. Your input value is first converted to seconds using the corresponding multiplier.
2. The base value in seconds is then converted to all other time units.

**Example:**

* Input: `2` hours → Base value in seconds = `2 × 3600 = 7200 s`
* Converted: `7200 s = 2 hours = 120 minutes = 7,200,000 ms`

This ensures accurate and consistent conversions across all units.

---

### Supported Units

The converter supports a comprehensive range of time units:

* **Milliseconds (ms)** – smallest practical unit in this tool
* **Seconds (s)** – standard base unit
* **Minutes (minute)** – 60 seconds
* **Hours (hour)** – 60 minutes
* **Days (day)** – 24 hours
* **Weeks (week)** – 7 days
* **Months (month)** – average 30.44 days
* **Years (year)** – 365.25 days (accounting for leap years)
* **Centuries (century)** – 100 years
